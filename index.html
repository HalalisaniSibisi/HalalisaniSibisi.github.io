02 · Projects

What I've Built

LushBites Food Truck Management System Completed

A multi-screen food truck management system enabling customers to browse menus, place orders, and submit personal details through a structured WinForms interface. Features South African ID validation using the Luhn algorithm — the same real-world verification logic used in financial and insurance systems. Order and customer data persists to file and is retrieved dynamically into a DataGridView for searchable transaction records.

C# WinForms File Persistence Luhn Algorithm
Grocery Ordering Management System Phase 2 In Progress

Addresses a real operational problem at an old age home where a manual, paper-based grocery ordering process resulted in fragmented records and limited reporting. Built a structured SQL Server database system with ERD design, role-based authentication separating resident and Matron/admin access, an admin dashboard for member and item management, and soft-delete to preserve historical data, improving data accuracy and establishing a scalable foundation for management reporting.

C# WinForms SQL Server Role-Based Auth ERD Design Soft Delete
Student Registration System Phase 1 Complete

Replaced a manual, queue-based student registration process with a C# (.NET) Windows Forms system backed by SQL Server. Students can register digitally while administrators manage courses and records through a structured platform. Built with a strong focus on database design (ERD), CRUD operations, role-based authentication, and a clean data access layer.

C# .NET WinForms SQL Server ERD CRUD
Cloud Resume Challenge In Progress

Personal portfolio hosted on GitHub Pages and integrated with Azure Functions, Cosmos DB, and GitHub Actions CI/CD. Built to demonstrate cloud engineering, serverless computing, infrastructure automation, and modern deployment practices.

GitHub Pages Azure Functions Cosmos DB JavaScript GitHub Actions REST API
CafeOps Insight System In Development

A multi-layer café management and analytics platform bridging the gap between daily operations and real business visibility. Implements a separated architecture: a C# POS system (SQL Server OLTP) for live transaction capture, and a full analytics pipeline using Python, Apache Spark, and Azure Data Lakehouse (Bronze/Silver/Gold medallion architecture) to transform raw data into structured business intelligence, surfaced through Power BI dashboards for branch managers and owners across multiple locations.

C# ASP.NET SQL Server Python Apache Spark Azure Data Lakehouse Power BI
